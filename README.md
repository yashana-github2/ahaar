Aahaar 🍽️

Voice-Enabled Accessible Food Ordering Web Application

Aahaar is an accessibility-focused food ordering web application designed to help visually impaired users order food independently through voice interaction and audio guidance. The system combines speech recognition, responsive design, and real-time order workflow simulation to create an inclusive and user-friendly food delivery experience.


🚀 Features

* 🎤 Voice-enabled navigation and ordering
* 🔊 Audio feedback using Speech Synthesis
* 🔐 OTP-based login and authentication
* 🍕 Restaurant and menu browsing
* 🔎 Smart food search functionality
* 🛒 Cart and checkout system
* 💳 Simulated payment workflow
* 🚚 Delivery partner assignment and tracking
* 📦 Real-time order status updates
* ✅ Accessibility-focused form validation
* 📱 Responsive UI design



🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* MySQL
* Deepgram API (for voice)



🎙️ Voice Accessibility

Aahaar uses browser-based Web Speech APIs to improve accessibility for visually impaired users.

Technologies Used

- SpeechRecognition API — Converts speech into text
- SpeechSynthesis API — Provides spoken feedback
- Deepgram API — Enhances speech recognition and voice processing


Example Voice Commands

* “Search pizza”
* “Add burger to cart”
* “Go to checkout”
* “Place order”



⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/Aahaar.git
```

2. Open the project folder

3. Run using:

* VS Code Live Server
  or
* XAMPP/WAMP localhost

4. Allow microphone permission in browser for voice features.

🎯 Objective

The primary objective of Aahaar is to make online food ordering more accessible and inclusive for visually impaired users through voice-guided interaction and accessible web design.
